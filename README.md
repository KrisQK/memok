# Memok

轻量、好用的本地记事与清单小工具 — 速度快、离线可用、基于现代前端技术，适合随手记录想法与任务。

## 为什么选择 Memok

- 极简：单文件静态页面（HTML + Vue3）即开即用，无复杂依赖。
- 本地优先：数据存储在浏览器的 localStorage，隐私友好，断网也能用。
- 交互友好：支持回车添加、拖拽排序（SortableJS）、勾选完成与删除。
- 轻量美观：使用 Pico.css 做基础样式，界面清爽，移动/桌面都适配。

## 主要功能（卖点）

- 快速记事：输入回车即可添加。
- 拖拽排序：长按或拖动列表项改变顺序，顺序会持久化。
- 完成标记：勾选任务会有划线样式并保存状态。
- 单文件部署：把整个目录放到任意静态服务器即可（Nginx、Apache、Python 简易服务器等）。

## 目录结构

项目根（你的文档根）示例：

```
/Users
  ├─ index.html   # 主应用
  └─ README.md
```
